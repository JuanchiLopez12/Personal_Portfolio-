Dining Review API

This project is designed to provide the user with a list of dinings, their reviews and all of their information.
It allows users to write a review which will afterwards be approved or denied by a admin of the system.
It utilizes a embedded database for testing purposes but it can be scaled with either SQL or MySql for bigger purposes.

  • Utilized Java for the entity design.

  • Spring was used to provide the mapping of the API and transform the entities into database tables.
